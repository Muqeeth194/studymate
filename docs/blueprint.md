# **App Name**: StudyVerse AI

## Core Features:

- Personalized Roadmap Generation: Generate a tailored learning roadmap based on user preferences and uploaded documents, leveraging the Claude API as a tool.
- Voice-First Learning: Enable voice-based Q&A and spoken explanations using Whisper API for transcription and ElevenLabs for text-to-speech.
- Adaptive Quiz System: Create dynamic quizzes that adjust difficulty based on user performance, powered by Claude API as a tool.
- Document Upload and RAG: Allow users to upload study materials and utilize RAG with Pinecone to provide context-aware answers via Claude API.
- Progress Tracking and Analytics: Display user progress, quiz performance, and weak areas on a dashboard.
- User Authentication and Preference Collection: Allow users to sign up, log in and save user data in the MongoDB database
- Gamification Elements: Implement streaks, points, and achievements to encourage continued engagement and give awards to user performance.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to convey intellect, focus, and sophistication, complementing the academic nature of the application.
- Background color: Light lavender (#EDE7F6) for a soft, non-distracting backdrop, ensuring comfortable readability and prolonged screen time.
- Accent color: Vibrant purple (#9575CD) to highlight interactive elements and calls to action, providing a clear visual distinction and engaging user attention.
- Body font: 'Inter', sans-serif, for a clean and modern look that ensures readability across devices.
- Headline font: 'Space Grotesk', sans-serif, is chosen for headlines to give a computerized feel that will contrast the body font
- Use minimalist icons to represent different learning topics and features, ensuring clarity and visual appeal.
- Implement a clear, structured layout with distinct sections for roadmap, study sessions, and analytics, providing an intuitive user experience.
